Mr.Memo 개인정보처리방침

최종 업데이트: 2026-05-06 / 개발자: Beachside Studio

소개

Mr.Memo(이하 "본 앱")는 Beachside Studio(이하 "당사")가 제공하는 메모 애플리케이션입니다. 본 방침은 본 앱이 어떤 정보를 다루며 어떻게 보호하는지 설명합니다.

1. 사용자가 입력한 메모 등의 처리

메모, 제목, 태그, 폴더 이름, 색상 라벨 등 본 앱에서 사용자가 입력하거나 작성한 모든 데이터는 사용자 기기 내부의 SQLite 데이터베이스에만 저장됩니다.

  • 이러한 데이터는 외부 서버로 전송되지 않습니다
  • 당사(Beachside Studio)는 이러한 데이터에 접근할 수 없습니다
  • 광고주를 포함한 어떤 제3자에게도 제공되지 않습니다

2. 백업 기능

본 앱의 백업 기능은 메모 데이터를 암호화된 파일(확장자 .mrmemo)로 기기 내부에 생성합니다.

  • 암호화 방식: AES-256-CBC + HMAC-SHA256(Encrypt-then-MAC 구성)
  • 키 도출: PBKDF2-SHA256(반복 횟수 15,000회)
  • 사용자가 설정한 비밀번호 없이는 복원할 수 없습니다
  • 백업 파일은 시스템 공유 시트를 통해 사용자가 선택한 대상(Gmail, Google Drive, 기기 저장소 등)으로 전달됩니다
  • 당사는 자동으로 클라우드에 업로드하지 않습니다

3. 광고에 대해

본 앱은 무료로 제공되며, Google AdMob을 통한 광고로 운영됩니다.

3.1 광고 위치

  • 메모 목록 화면의 헤더 바로 아래에 배너 광고 표시
  • 새 메모를 5개 작성할 때마다 1회 전체 화면 광고 표시
  • 편집 화면, 설정, 즐겨찾기, 폴더 등 다른 화면에는 광고를 표시하지 않습니다

3.2 AdMob이 수집하는 정보

광고 게재를 위해 Google AdMob은 다음과 같은 정보를 수집할 수 있습니다:

  • 광고 ID(GAID / Google Advertising ID)
  • App Set ID(Android 12 이상)
  • IP 주소로 추정되는 대략적인 위치(국가 수준)
  • 기기 유형, OS 버전, 화면 크기
  • 광고 노출, 클릭 등 상호작용

이러한 정보는 Google의 광고 네트워크로 전송되며 당사와 공유되지 않습니다. Google의 처리에는 Google 개인정보처리방침이 적용됩니다.

3.3 메모 내용과 광고의 경계

사용자가 본 앱에 입력한 메모, 제목, 태그 등의 내용은 광고 시스템(Google AdMob 포함)에 공유되지 않습니다. 코드 상으로 메모 기능과 광고 기능은 명확하게 분리되어 있습니다.

3.4 동의 관리(EU, 영국, 스위스)

EU, EEA, 영국, 스위스의 사용자는 본 앱을 처음 시작할 때 Google User Messaging Platform (IAB TCF v2.2) 동의 대화 상자가 표시됩니다. 동의 상태에 따라 개인 맞춤 광고 또는 비개인 맞춤 광고가 표시됩니다.

3.5 기본 설정 및 사용자 제어

  • 본 앱의 기본 설정은 "비개인 맞춤 광고"(행동 추적이 없는 광고)입니다
  • "설정 / 정보 / 광고 개인정보 옵션"에서 언제든지 동의 상태를 다시 확인하거나 변경할 수 있습니다
  • 기기의 Google 설정에서 "광고 ID 재설정"을 실행하여 광고 ID를 변경할 수도 있습니다

4. 수집하지 않는 정보

본 앱은 다음 정보를 일절 수집하지 않습니다(해당 하드웨어 권한 자체를 요청하지 않습니다):

  • 카메라, 마이크, 사진 라이브러리 접근
  • 연락처, 통화 기록, SMS
  • GPS 또는 정밀 위치 정보(AdMob의 IP 기반 국가 추정 제외)
  • Google 계정 정보
  • 자체 사용 분석 또는 충돌 보고서(Firebase Analytics / Sentry / Crashlytics 등은 통합되지 않음)

5. 데이터 보관 및 삭제

  • 본 앱의 데이터는 사용자가 앱을 제거하거나 앱 내 기능으로 삭제할 때까지 기기 내부에 보관됩니다
  • 당사 서버에 데이터를 보관하지 않습니다
  • AdMob을 통해 Google로 전송된 데이터의 삭제에 대해서는 Google Play 데이터 삭제 절차를 따라주십시오

6. 어린이 개인정보

본 앱은 13세 이상 사용자를 대상으로 합니다. AdMob 광고의 특성상, 어린이(COPPA 대상)를 위해 설계되지 않았습니다.

7. 방침 변경

본 방침은 기능 추가 또는 법적 변경에 따라 개정될 수 있습니다. 중요한 변경이 있는 경우 본 페이지의 "최종 업데이트" 날짜를 갱신합니다.

8. 문의

본 방침에 관한 질문은 Google Play Console에 등록된 개발자 연락처 이메일로 문의해 주십시오.
